# Hướng dẫn sử dụng — keyword-ggads

## Khi nào dùng skill này?
- Khi bắt đầu setup chiến dịch Google Search Ads mới
- Khi cần mở rộng bộ từ khóa cho campaign đang chạy
- Khi muốn kiểm tra Search Volume trước khi phân bổ ngân sách

## Cách dùng

1. Gõ `/keyword-ggads` trong Claude Code
2. Cung cấp danh sách nhóm sản phẩm/dịch vụ cần research
3. Xác nhận khu vực địa lý (mặc định: toàn quốc)
4. Nhận bảng kết quả → copy sang Google Sheet

## Ví dụ input

```
Sản phẩm: Tủ lạnh Samsung
Nhóm cần research:
- Tủ lạnh Samsung chung
- Tủ lạnh Samsung inverter
- Tủ lạnh Samsung 2 cánh
Khu vực: Toàn quốc (VN)
```

## Lưu ý quota KeywordTool
- Mỗi lần check quota = 1 lần dùng
- Ưu tiên lấy đúng keywords phù hợp ngay từ đầu
- Xem file `sample-output.md` để biết format kết quả
